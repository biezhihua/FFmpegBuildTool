
# Init Repository 初始化库 

```
./init-ffmpeg.sh armv7a | all
```

# Clean Repository Cache 清理库缓存

```
./init-ffmpeg.sh clean
```

# Config Build Params 配置构建参数

```
./init-config min | default | mp4 or ...
```

# Build Library 构建库

```
./compile-ffmpeg.sh armv7a | all
```

# Check Output 查看输出结果

```
open ./product
```
